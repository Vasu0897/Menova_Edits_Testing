# MeNova Health — Make.com Setup Guide
## Google Docs Template + Webhook Integration

---

## Overview

This guide walks you through setting up Make.com to:
1. Receive quiz data from your website via webhook
2. Create a formatted Google Doc for each patient assessment
3. Save it to Google Drive
4. (Optional) Send confirmation emails

---

## Part 2: Set Up Make.com Scenario

### Step 1 — Open your Make.com scenario

1. Go to [make.com](https://make.com)
2. Open the scenario you created earlier (the one with the webhook)
3. You should see: **[Webhook] → [empty space for next module]**

### Step 2 — Add the Google Docs module

1. Click the **+** button after the Webhook
2. Search for **Google Docs**
3. Select **Create a Document from a Template**
4. Click **Add**

### Step 3 — Connect your Google account

1. Click **Connection** dropdown
2. Click **Create a new connection**
3. Sign in with your Google account (the one that owns your Drive)
4. Grant Make.com permission to access Google Docs and Drive
5. Click **Save**

### Step 4 — Configure the Google Docs module

Now you'll see a form with these fields:

| Field | What to enter |
|---|---|
| **Template Document ID** | Paste the document ID you copied above |
| **Document Title** | `{{name}} — Assessment — {{timestamp}}` |
| **Folder ID** | (See Step 5 below) |

### Step 5 — Get your Google Drive folder ID

1. Go to [Google Drive](https://drive.google.com)
2. Create a new folder called **"Patient Assessments"**
3. Open the folder
4. Copy the folder ID from the URL:
   - URL: `https://drive.google.com/drive/folders/1abc2def3ghi4jkl5mno6pqr7stu8vwx`
   - Copy: `1abc2def3ghi4jkl5mno6pqr7stu8vwx`
5. Paste this into the **Folder ID** field in Make.com

### Step 6 — Map the webhook fields to template placeholders

In the Google Docs module, scroll down to **Variables** section. You'll see a list of all the `{{placeholder}}` fields from your template.

For each placeholder, click the field and select the corresponding webhook data:

| Template Placeholder | Webhook Field to Map |
|---|---|
| `{{name}}` | `name` |
| `{{email}}` | `email` |
| `{{timestamp}}` | `timestamp` |
| `{{score}}` | `score` |
| `{{maxScore}}` | `maxScore` |
| `{{tier}}` | `tier` |
| `{{answers_stage}}` | `answers` → `stage` (first item) |
| `{{answers_hot_flashes}}` | `answers` → `hot_flashes` (first item) |
| `{{answers_sleep}}` | `answers` → `sleep` (first item) |
| `{{answers_mood}}` | `answers` → `mood` (first item) |
| `{{answers_brain_fog}}` | `answers` → `brain_fog` (first item) |
| `{{answers_symptoms_multi}}` | `answers` → `symptoms_multi` (join with ", ") |
| `{{answers_impact}}` | `answers` → `impact` (first item) |
| `{{answers_treatment_history}}` | `answers` → `treatment_history` (first item) |
| `{{recommendation}}` | `recommendation` |

**Mapping example for `{{answers_stage}}`:**
1. Click in the `{{answers_stage}}` field
2. A popup appears showing available data
3. Click **answers** → **stage**
4. If it's an array, select the first item `[0]`
5. Click **OK**

## Part 3: (Optional) Add Email Notifications

### Step 1 — Add Gmail module

1. Click **+** after the Google Docs module
2. Search for **Gmail**
3. Select **Send an Email**
4. Click **Add**

### Step 2 — Configure the email

Set these fields:

| Field | Value |
|---|---|
| **To** | `{{email}}` (the patient's email from webhook) |
| **Subject** | `Your MeNova Health Assessment Results — {{name}}` |
| **Body** | See template below |

{{recommendation}}

Questions?
Email: hello@menovahealth.ca

2. Add another **Gmail** module
3. Send to: **your email address**

## Part 4: Test the Scenario

### Step 1 — Enable the scenario

1. In Make.com, click the toggle switch to turn on the scenario
2. Set it to run **Immediately** (real-time)

### Step 2 — Test with the quiz

1. Go to your MeNova Health website
2. Click **"Take the Free Symptom Quiz"**
3. Answer all 8 questions
4. Enter a test name and email
5. Click **"View My Results"**

### Step 3 — Check the results

1. Go to [Google Drive](https://drive.google.com)
2. Open the **"Patient Assessments"** folder
3. You should see a new document with the name: `[Your Test Name] — Assessment — [timestamp]`
4. Open it and verify all fields are filled correctly
5. Check your email for the confirmation message

## Troubleshooting

| Problem | Solution |
|---|---|
| **Document not created** | Check Make.com execution history (click the scenario, then "Execution history"). Look for error messages. |
| **Fields showing `{{placeholder}}`** | The mapping wasn't set correctly. Go back to the Google Docs module and re-map each field. |
| **Email not sent** | Verify Gmail connection is authorized. Check spam folder. |
| **Wrong data in fields** | Make sure you're selecting the correct webhook field (e.g., `answers.stage[0]` not just `answers.stage`) |
| **Folder ID not working** | Copy it again from the URL. Make sure you're in the correct folder. |

## Field Reference: What the webhook sends

When a patient submits the quiz, Make.com receives this JSON:

```json
{
  "name": "Sarah Johnson",
  "email": "sarah@example.com",
  "timestamp": "2026-04-06T12:34:56.789Z",
  "score": 24,
  "maxScore": 48,
  "tier": "Moderate",
  "answers": {
    "stage": ["peri_mid"],
    "hot_flashes": ["moderate"],
    "sleep": ["moderate"],
    "mood": ["mild"],
    "brain_fog": ["moderate"],
    "symptoms_multi": ["joint_pain", "weight_gain", "dry_skin"],
    "impact": ["work"],
    "treatment_history": ["no"]
  },
  "recommendation": "Based on your assessment, you're experiencing moderate menopause symptoms that are affecting your daily life. A personalized Bioidentical Hormone Replacement Therapy (BHRT) plan could help you regain your energy and quality of life..."
}
```

## Next Steps After Setup

1. **Test thoroughly** — submit 3–5 test assessments to ensure everything works
2. **Customize the Google Doc template** — add your clinic logo, colors, or additional sections
3. **Add a follow-up automation** — after 3 days, if the patient hasn't booked, send a reminder email
4. **Monitor submissions** — check your admin email weekly to see who's taking the quiz and their severity tiers
5. **Iterate** — after 10 real patients, review the docs and adjust the template if needed
