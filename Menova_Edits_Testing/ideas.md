# MeNova Health — Design Brainstorm

## Context
A premium D2C menopause telehealth platform for Canadian women aged 40–60 in Vancouver. Modeled after MEDVi's layout: dark forest-green hero, horizontal category scroll, feature sections, testimonials, pricing, FAQ.

<idea>

**Design Movement:** Biophilic Wellness Modernism

**Core Principles:**
1. Warm earth tones layered with deep botanical greens — nature as medicine
2. Asymmetric editorial layouts that feel like a premium health magazine
3. Generous whitespace with deliberate tension between text and imagery
4. Humanistic photography: real women in their 40s–50s, outdoors, joyful

**Color Philosophy:**
- Primary: Deep Forest Green `#1A3C2E` (authority, nature, healing)
- Accent: Warm Terracotta `#C4714A` (vitality, femininity, warmth)
- Surface: Cream `#FAF7F2` (softness, premium, approachable)
- Text: Charcoal `#2D2D2D`
- Highlight: Sage `#7BA68A` (secondary green for gradients)

**Layout Paradigm:**
- Full-width dark green hero with large serif headline + small sans-serif subtext
- Horizontal scrolling category cards (like MEDVi)
- Alternating left/right content blocks with overlapping image/text
- Sticky navigation with transparent-to-solid scroll transition

**Signature Elements:**
1. Leaf/botanical SVG dividers between sections
2. Circular image crops with soft drop shadows
3. Pill-shaped CTA buttons in terracotta

**Interaction Philosophy:**
- Smooth fade-in on scroll for each section
- Hover states: subtle scale + shadow lift on cards
- CTA buttons: color fill transition on hover

**Animation:**
- Hero text: staggered word reveal (0.1s delay per word)
- Section entrance: translateY(20px) → 0 with opacity 0→1
- Category carousel: smooth horizontal scroll with momentum

**Typography System:**
- Display: Playfair Display (serif, bold) — headlines
- Body: DM Sans (humanist sans) — paragraphs, UI
- Accent: DM Sans Italic — pull quotes, taglines

## Selected Design: **Biophilic Wellness Modernism**

Deep Forest Green hero, Warm Terracotta CTAs, Cream surfaces, Playfair Display headlines, DM Sans body. Asymmetric editorial layout with botanical motifs and smooth scroll animations.
